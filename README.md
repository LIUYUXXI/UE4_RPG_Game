# UE4_RPG_Game（更新中）
**UE4制作的第三人称RPG游戏**
<br><br>参考视频教程：https://www.sikiedu.com/my/course/295
## 已实现功能：
* 1、第三人称视角人物移动和跳跃、移动动画切换；
* 2、小地图、头像、血量、技能栏和菜单栏的UI
* 3、鼠标点击自动寻路功能
* 4、鼠标控制摄像机视野360度转换
